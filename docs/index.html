410-A1 Design Document

A design document for a game named Cybersky, the development title for a fairly simplistic shmup mobile game I've come up with revolving around roguelite features. This document outlines the design and philosophy of the game, acting as a how-to on how to make it.

Skills Utilised:

LaTeX · Game Design

View the document!

435-A1 Website

A near scratch-written website for the CI435 Assignment 1 brief, designed to show the learning progress made over the duration of the module so far. This site uses a few add-in modules for scrollspy and text highlighting, but is otherwise entirely handwritten by myself.

It demonstrates my skills in HTML and CSS, though doesn't demonstrate much of my JS or Bootstrap knowledge. The site is responsive, using a sidebar in widescreen viewports and having the sidebar at the top of the page on mobile or thin viewports.

Skills Utilised:

HTML · CSS

View the site!

435-A1 Ubisoft Research Video

This video was produced as part of a deep-dive into Ubisoft, the games producer and business owning multiple development studios. I was a writer and lead director on this project, using my skills with photography and lighting to produce the final project.

This is a group project, and as such is not wholly my own work. This project sadly was delayed past the deadline due to some technical issues, so the final product is somewhat rushed. In spite of this, it's an effective video, if a little amateurish.

Skills Utilised:

Photography · Set Lighting · Script Writing · Research
